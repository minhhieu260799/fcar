# FCAR Logo

Logo da thiet ke o dang vector:

- `fcar-logo.svg` (ban day du icon + chu + slogan)
- `fcar-logo-horizontal.svg` (ban ngang khong slogan)
- `fcar-logo-icon.svg` (ban icon vuong cho favicon/app icon)
- `fcar-logo-mono-dark.svg` (ban don sac den de in/khac)
- `fcar-logo-mono-light.svg` (ban don sac trang cho nen toi)

## Mau su dung de nghi

- Primary blue: `#0F6CBD`
- Deep blue: `#0A4F8A`
- Text dark: `#0A1F33`

## Cach dung nhanh

- Website:
  - Header/logo chinh: `fcar-logo-horizontal.svg`
  - Favicon/app icon: `fcar-logo-icon.svg`
- Bao cao/slide: `fcar-logo.svg` hoac `fcar-logo-horizontal.svg`
- In an:
  - Nen sang: `fcar-logo-mono-dark.svg`
  - Nen toi: `fcar-logo-mono-light.svg`
- Word/PowerPoint: Insert -> Pictures -> chon `fcar-logo.svg`.
- In an: uu tien SVG/PDF de khong bi vo hat.

## Luu y

- Khuyen nghi khong keo gian logo theo 1 chieu de tranh meo.
- Giu khoang trong toi thieu xung quanh logo bang it nhat chieu cao chu "F".
