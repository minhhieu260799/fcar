# FCAR Mockups

Bo nay gom 8 mockup HTML tinh cho cac giao dien chinh:

1. `01-home.html`
2. `02-login.html`
3. `03-cars-list.html`
4. `04-car-detail.html`
5. `05-deposit-payment.html`
6. `06-account-history.html`
7. `07-admin-orders.html`
8. `08-admin-dashboard.html`

## Cach xem

- Mo file `index.html` trong browser.
- Hoac mo truc tiep tung file `.html`.
- Logo dung chung dat tai `assets/logo-fcar.svg`.

## Che do clean cho luan van

- Mac dinh: giao dien brand FCAR.
- Ban clean: them `?theme=thesis` vao URL.
- Vi du: `01-home.html?theme=thesis`

## Cach xuat PNG de dua vao bao cao

1. Mo tung mockup trong Chrome/Edge.
2. Nhan `Ctrl + Shift + I` -> Toggle device toolbar (neu can).
3. Nhan `Ctrl + Shift + P` -> go `Capture full size screenshot`.
4. Luu anh PNG.

Meo: Dung zoom browser 100% de bo cuc dep va dong nhat giua cac man.
